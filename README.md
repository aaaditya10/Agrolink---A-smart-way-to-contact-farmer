# Agrolink---A-smart-way-to-contact-farmer
A role-based agricultural trading platform connecting farmers, buyers, retailers, and logistics partners. Features direct crop listings, order management, real-time messaging, and simulated digital payments — built using HTML, CSS, and JavaScript.

# AgroLink — Frontend Prototype

Simple frontend prototype for AgroLink (HTML, CSS, JS).
Features: role-based signup/login (OTP simulated), listings, marketplace, chat (BroadcastChannel realtime simulation), payments (simulated), admin panel.

## Run locally
1. Open `index.html` in browser, OR
2. Run a local server:
   - `python -m http.server 8000` then open http://localhost:8000

## Notes
- This is frontend-only demo. Do NOT store real personal/sensitive data here.
- Replace simulated OTP and localStorage with secure backend before production.
